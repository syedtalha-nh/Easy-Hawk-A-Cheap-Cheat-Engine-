🦅 EasyHawk

A beginner-friendly memory scanner for Windows — easier than Cheat Engine.

⚠️ Note: This project is not fully completed and is still in the testing phase. Bugs may occur.

✨ What it does:
Search for values in any running process (health, gold, ammo, etc.)
Narrow down results with First Scan → Next Scan
Edit and freeze values directly in memory


🚀 How to run:

bash# Run directly
python easyhawk.py

# Or build an exe
python -m PyInstaller --onefile --windowed --name EasyHawk easyhawk.py

Always run as Administrator for memory access to work.

⚠️ Disclaimer
For single-player and educational use only. Do not use in online games.
